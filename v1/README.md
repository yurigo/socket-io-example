# Socket.io Example

Este proyecto consta de cliente y servidor.

Se utiliza el cliente y el servidor de socket.io

<br>

## Para ejecutar el servidor

Entramos en la carpeta del servidor
```
cd server
```

Instalamos las dependencias
```
npm i
```

Ejecutamos el servidor
```
npm start
```

<br>
